# Arkvark - Bitcoin Wallet based on Arkade

## Project Overview

Arkvark is a React Native bitcoin wallet application for [Arkade](https://docs.arkadeos.com/) built with Expo SDK 54 and expo-router for file-based routing. It targets iOS, Android, and web. The New Architecture is enabled (`newArchEnabled: true`) along with the React Compiler experiment.
